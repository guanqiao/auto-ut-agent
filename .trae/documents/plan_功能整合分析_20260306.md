### 2.4 使用场景

#### 场景 1: 测试失败聚类

在生成测试后，如果有测试失败，聚类相似失败：

```python
async def _cluster_test_failures(self, failures: List[TestFailure]):
    """聚类相似测试失败"""
    if not self.smart_clusterer:
        return failures
    
    clusters = self.smart_clusterer.cluster_failures(failures)
    # 只对每个聚类调用一次 LLM
    for cluster in clusters:
        await self._handle_failure_cluster(cluster)
```

#### 场景 2: 相似代码识别

在生成新测试前，识别项目中相似代码：

```python
async def find_similar_code(self, target_code: str) -> List[CodeSnippet]:
    """查找相似代码"""
    if not self.smart_clusterer:
        return []
    
    # 使用语义相似度查找
    return self.smart_clusterer.find_similar(target_code)
```

### 3.3 集成位置

**方案 A**: 集成到 ToolOrchestrator

**文件**: `pyutagent/agent/tool_orchestrator.py`

```python
class ToolOrchestrator:
    def __init__(self, ...):
        self.tool_validator = create_tool_validator()
    
    async def execute_tool(self, tool_name: str, **kwargs):
        # 执行前验证
        result = self.tool_validator.validate_tool_call(
            tool_name, (), kwargs
        )
        
        if not result.valid:
            # 处理验证失败
            for issue in result.issues:
                if issue.severity in (ERROR, CRITICAL):
                    raise ValidationError(issue.message)
        
        # 执行工具
        return await self._execute_tool(tool_name, **kwargs)
```

**方案 B**: 集成到 EnhancedAgent

**文件**: `pyutagent/agent/enhanced_agent.py`

```python
class EnhancedAgent:
    async def _execute_tool_safely(self, tool_name: str, **kwargs):
        # 使用 tool_validator 验证
        if self.tool_validator:
            result = self.tool_validator.validate_tool_call(
                tool_name, (), kwargs
            )
            
            if not result.valid:
                logger.warning(f"Tool validation issues: {result.issues}")
                
                # 记录但不阻止执行
                for issue in result.warnings:
                    logger.warning(f"Warning: {issue}")
        
        # 执行工具
        return await self._execute_tool(tool_name, **kwargs)
```

@click.command(name='voice')
@click.option('--listen', is_flag=True, help='Listen for voice input')
@click.option('--speak', type=str, help='Text to speak')
def voice_command(listen: bool, speak: str):
    """Voice interaction commands."""
    from pyutagent.agent.voice import VoiceInputHandler, VoiceOutputHandler
    
    if listen:
        handler = VoiceInputHandler()
        text = handler.listen()
        console.print(f"[cyan]You said: {text}[/cyan]")
    
    if speak:
        output = VoiceOutputHandler()
        output.speak(speak)
