# PyUT Agent GUI 增强计划

## 项目概述
当前GUI基于PyQt6构建，是一个功能完整的AI单元测试生成工具。本计划旨在通过现代化的UI设计、增强的交互体验和新增功能来全面提升用户体验。

---

## 阶段一：UI/UX 现代化改进

### 1.1 主题与样式系统
**目标**: 建立现代化的视觉设计系统

**任务**:
- [ ] 创建统一的样式管理器 (`ui/styles/style_manager.py`)
  - 支持亮色/暗色主题切换
  - 定义颜色系统（主色、辅助色、状态色）
  - 定义字体系统（标题、正文、代码）
  - 定义间距和圆角规范
- [ ] 创建主题配置文件 (`ui/styles/themes/`)
  - `light_theme.json` - 浅色主题
  - `dark_theme.json` - 深色主题
- [ ] 更新所有组件使用样式管理器
  - 主窗口、对话框、组件统一应用
- [ ] 添加主题切换功能（设置菜单）

**设计规范**:
```
主色调: #2196F3 (Material Blue)
成功色: #4CAF50 (Green)
警告色: #FF9800 (Orange)
错误色: #F44336 (Red)
背景色: #FAFAFA (Light) / #1E1E1E (Dark)
卡片背景: #FFFFFF (Light) / #2D2D2D (Dark)
圆角: 8px (卡片), 4px (按钮)
```

### 1.2 主窗口布局优化
**目标**: 提升空间利用率和视觉层次

**任务**:
- [ ] 重新设计三栏布局
  - 左侧面板：项目树（可折叠，宽度可调）
  - 中间面板：聊天区域（自适应宽度）
  - 右侧面板：进度和日志（可切换标签页）
- [ ] 添加工具栏（快速操作按钮）
  - 打开项目、生成测试、批量生成、配置
- [ ] 优化状态栏
  - 添加连接状态指示器
  - 添加当前项目显示
  - 添加操作快捷提示
- [ ] 实现面板折叠/展开功能

### 1.3 聊天界面增强
**目标**: 提升对话体验和可读性

**任务**:
- [ ] 重新设计消息气泡
  - 更现代的样式（阴影、圆角）
  - 时间戳显示
  - 复制消息功能
- [ ] 增强代码块显示
  - 语法高亮（使用QSyntaxHighlighter）
  - 代码折叠功能
  - 复制代码按钮
  - 行号显示
- [ ] 添加消息操作菜单
  - 重新生成、编辑、删除
- [ ] 支持Markdown渲染
  - 表格、列表、链接
- [ ] 添加输入框增强
  - 多行输入支持（Shift+Enter换行）
  - 输入历史（上下箭头）
  - 快捷命令提示（/help等）

## 风险评估与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 重构引入bug | 高 | 小步重构，频繁测试，保持向后兼容 |
| 性能下降 | 中 | 性能基准测试，渐进优化 |
| 用户不适应 | 低 | 保留经典模式选项，提供切换 |
| 开发时间超期 | 中 | 分阶段交付，优先核心功能 |
